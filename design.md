# BR Jalan Securities 网站设计方案

## 设计理念

### 核心概念
- **专业可信**: 体现金融行业的权威性和可靠性
- **现代简约**: 清晰的信息层次和直观的用户体验
- **数据驱动**: 突出数据可视化和分析能力
- **印度本土化**: 融入印度金融市场的特色元素

### 颜色方案
- **主色调**: 深蓝色 (#1e3a8a) - 代表信任、稳定、专业
- **辅助色**: 金色 (#f59e0b) - 象征财富、成功、繁荣
- **中性色**: 浅灰 (#f8fafc) 和深灰 (#374151) - 确保内容可读性
- **强调色**: 绿色 (#10b981) - 代表增长、收益
- **警告色**: 红色 (#ef4444) - 用于风险提示

### 字体设计
- **标题字体**: Playfair Display (衬线字体) - 优雅、权威
- **正文字体**: Inter (无衬线字体) - 现代、清晰、易读
- **数据字体**: JetBrains Mono (等宽字体) - 用于数字和代码显示

### 布局原则
- 12列网格系统，确保内容对齐
- 充足的留白空间，避免信息过载
- 卡片式设计，模块化内容展示
- 响应式布局，适配各种设备

## 视觉效果

### 背景效果
- **主页**: 使用 shader-park 创建动态金融数据流背景
- **内页**: 使用 p5.js 生成抽象的几何图案背景
- **渐变**: 深蓝到金色的微妙渐变，营造专业氛围

### 动画效果
- **滚动动画**: 使用 Anime.js 实现元素淡入和位移动画
- **数据可视化**: ECharts.js 创建交互式图表和仪表盘
- **文字效果**: Typed.js 实现打字机效果的标题动画
- **粒子效果**: 使用 matter.js 创建金币飘落效果

### 图标设计
- 使用 lucide-react 图标库
- 统一的线条风格和颜色
- 金融主题图标：图表、趋势、投资组合等

### 图像风格
- 高质量的商业和金融主题图片
- 统一的色调处理，与品牌色彩协调
- 专业的商务人士和金融场景

## 交互设计

### 悬停效果
- 卡片轻微上升和阴影加深
- 按钮颜色渐变和边框发光
- 图片轻微缩放和遮罩显示

### 加载动画
- 渐进式内容加载
- 数字计数动画效果
- 图表绘制动画

### 响应式设计
- 移动端优先的设计思路
- 触摸友好的交互元素
- 自适应的图表和布局

## 数据可视化

### 图表风格
- 统一的配色方案
- 清晰的标签和图例
- 交互式数据点
- 响应式图表尺寸

### 仪表盘设计
- 卡片式布局
- 关键指标突出显示
- 实时数据更新效果
- 用户友好的交互界面
